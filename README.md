# clearwhitespace

> A simple module to remove whitespace from a string

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i clearwhitespace --save-dev
```

## Usage

```js
var clearwhitespace = require('clearwhitespace');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
